Passtiche Instantly Generates Passbook Badges

Passtiche makes it easy to create badges linking to Passbook passes.

Already have a pass file? Just link to the file and the badge will appear.

If you don't already have passes, you can generate them from simple link attributes and third-party APIs will be used to get extra info.

Passtiche badges open a dialog that previews the pass. The pass can be downloadeded via SMS/email from any device and shared with friends.

Create Your Own Passes & Engage Customers

Setting up your own Passbook passes is difficult and highly technical.

Passtiche offers the easiest way to create your own Passbook passes.

Include the Passtiche javascript library on your website to create customized pass content simply by adding a few attributes to links.

Third-party APIs are used to make it especially easy to create new passes. Just specify a venue name and other info will be retrieved.

You can use the Passtiche Dashboard to manage time and location-based push notifications and customize the appearance of passes.
