# Passtiche Overview

Passtiche Enhances Websites & Apps
With Custom Add to Passbook Badges


Download My Pass

Passtiche is a website plugin that brings Passbook to your website or mobile app with just a couple lines of code.

Use Passtiche to schedule personalized alerts & reminders for customers
    How it Works
  • Passtiche auto-generates badges for listings or links to existing Passbook files
  • Visitors can download your passes via SMS or email and share them with friends
  • Listings passes are automatically enhanced with extra info from Yelp & Foursquare
  • Schedule personalized alerts & reminders to be sent as Passbook notifications
  • New   Badges supported on over a dozen web browsers and mobile devices.

Add Passtiche To Your Website

You can bring Passbook support to your site by adding the Passtiche Javascript code snippet and following the directions below to convert normal links into Add to Passbook badges.

Start by pasting the following code snippet to your website: 
<script {% if current_user %}account="{{ current_user.short_code }}"{% end %} src="http://passtiche.com/js"></script>
{% if current_user %}
This script tag uses code {{ current_user.short_code }} for the account of {{ current_user.display_name() }}
{% end %} This markup should be placed at the bottom of your website's HTML (or generated-HTML) structure, before the closing <body> tag. The sections below describe how to generate badges, or check out a couple of examples on JSBin.

Create Badges for Existing Passes

You can specify the location of an existing pass file to use the Passtiche download dialog and sharing features while using your own Passbook server or the Pass creation tool of your choice.

To generate a badge for a pass file, include the Passtiche javascript library on the page and link to your pass file like you normally would:

<a href="http://www.passtiche.com/demo.pkpass">Download My Pass</a>

Create Badges for New Passes

If you don't already have a pass created, you can use special data-pass attributes for Add to Passbook badges that link to automatically generated passes.

Here is an example of a link that specifies the name of a location with a data-pass-loc attribute:

<a data-pass-loc="Roxie Theater" data-pass-city="San Francisco" ></a>

Add Optional Event/Activity Info

Here's the markup and resulting badge for for a link containing data-pass attributes for an activity name and price.

<a data-pass-loc="21st Amendment" data-pass-name="Sample a Dozen Beers"
 data-pass-price="10" data-pass-city="San Francisco" ></a>

Location Data Attributes

Remember to prefix these attributes with data-pass-
  • loc - (required) Name of location
  • city - (required) Name of city containing location
  • name - (required) Name of activity/event
  • price - (required) Price of activity/event
  • description - (required) Description of activity/event
You can also use data-pass-id to specify a 4-character ID code for an existing pass.
No other attributes are required when an ID is specified.
Passtiche now also supports open standards!
You can use hCard and Open Graph in addition to data-pass attributes.

Customize Badges & Passes

  Configuring Badge Size:

You can set the data-pass-size attribute to xsmall (20px height), small (30px height), medium (default, 40px height), large (60px height), or xlarge (80px height).
You can also set data-pass-size to a custom pixel integer value to specify a custom badge height between 20-160 pixels.
